# 行业规范 — 录入格式

## 一、资料来源

### 国际标准

| 来源 | 网址 | 说明 |
|------|------|------|
| IPC 官网 | https://www.ipc.org | 电子制造行业最权威的国际标准，IPC-A-610、J-STD-001 等 |
| ISO | https://www.iso.org | ISO 9001（质量管理）、ISO 14001（环境管理） |
| IEC | https://www.iec.ch | 国际电工委员会，电子元件、PCB设计标准 |
| JEDEC | https://www.jedec.org | 半导体器件可靠性标准（温度循环、静电敏感度等） |

## 二、数据库结构（process_standard 表）

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| standard_code | 否 | 规范编号 | IPC-A-610H |
| standard_name | **是** | 规范名称 | 电子组件的可接受性 |
| section | 否 | 章节 | 第 5 章 — 焊接 |
| summary | 否 | 内容概要（200-500字） | 见模板 |
| tags | 否 | 标签，逗号分隔 | SMT,焊接,品质验收 |
| related_process | 否 | 关联工艺，逗号分隔 | 回流焊,波峰焊 |


## 三、填写模板

```yaml
- standard_code: "IPC-A-610H"
  standard_name: "电子组件的可接受性"
  section: "第 5 章 — 焊接"
  summary: |
    本规范规定了电子组装中焊接连接的可接受条件。涵盖通孔焊接、
    表面贴装焊接、端子焊接等类型。将缺陷分为三个等级：
    1 级（通用电子）、2 级（专用服务电子）、3 级（高性能电子）。
    每种焊接类型均给出目标条件、可接受条件、缺陷条件的图示和说明。
  tags: "SMT,焊接,品质验收,IPC标准"
  related_process: "回流焊,波峰焊,手工焊"
```

### 空模板（复制填写）

```yaml
- standard_code: ""
  standard_name: ""
  section: ""
  summary: |
    
  tags: ""
  related_process: ""
```


## 四、推荐收集的规范

| 优先级 | 规范编号 | 规范名称 | 适用工艺 |
|--------|----------|----------|----------|
| 高 | IPC-A-610 | 电子组件的可接受性 | 焊接验收 |
| 高 | IPC-J-STD-001 | 焊接的电气和电子组件要求 | 焊接工艺 |
| 高 | IPC-7711/7721 | 电子组件的返工与维修 | 返修 |
| 中 | GB/T 4588.3 | 印制板的设计和使用 | PCB 设计 |
| 中 | GB/T 4728 | 电气简图用图形符号 | 电路设计 |
| 中 | SJ/T 10533 | 电子设备制造防静电技术要求 | ESD 防护 |
| 中 | JEDEC J-STD-020 | 塑封集成电路湿度/回流焊敏感度 | 元器件存储与焊接 |
| 低 | IPC-6012 | 刚性印制板的鉴定与性能规范 | PCB 制造 |
| 低 | IPC-A-600 | 印制板的可接受性 | PCB 来料检验 |
| 低 | ISO 9001 | 质量管理体系 | 全流程品质 |


## 五、提交规范

1. 按上述 YAML 模板填写，保存为 `.yaml` 文件
2. 文件命名：`{规范编号}.yaml`（如 `IPC-A-610.yaml`、`IPC-J-STD-001.yaml`）
3. 放入 `docs/data` 目录
